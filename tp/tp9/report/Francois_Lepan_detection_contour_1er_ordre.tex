\section*{Introduction}

Dans ce rapport nous allons voir comment mettre en oeuvre la détection de contours par approche du premier ordre. Nous verrons l'utilité du calcul de la norme d'un gradient et différents cueillages permettant d'aboutir à la détection de contour.

\section{Seuillage de la norme d'un gradient}

Tout d'abord pour détecter un contour il faut effectuer certains calculs. Le premier est le calcul de la norme d'une image.

Après avoir calculé sa norme nous obtenons des valeurs pour les pixels de la norme. Analysons ces valeurs.

\subsection*{L'image obtenue est de profondeur 32 bits. Comment expliquer sa valeur minimale et sa valeur maximale ?}

La valeur minimale est égale à 0 et sa valeur maximale est égale à 904,896. Ces valeurs s'expliquent par le calcul de la norme. Sachant que les valeurs pour l'image convolutionY  (\emph{cf.}~Fig.~\ref{convolution} à droite) varient entre  -804 et 895 et pour l'image convolutionX (\emph{cf.}~Fig.~\ref{convolution} à gauche)  elles varient entre -844 et 788.

Donc après calcul ($\sqrt{p_x^2~+~p_y^2}$) on peut obtenir des valeurs qui oscillent entre 0 et 904,896 sur l'image de la norme (\emph{cf.}~Fig.~\ref{norme}).

Maintenant que nous avons obtenu l'image de la norme, on se rend bien compte qu'il y a beaucoup de pixels "bruit".

Un procédé pour les éliminer est le seuillage des valeurs de l'image de la norme du gradient.

\subsection*{Seuillage de la norme du gradient précédemment calculée}

Une première approche est de calculer un seuil global sur l'image.

\subsubsection*{Est-il possible de trouver ainsi un seuil global pour l'image qui mette en évidence les pixels contours de manière satisfaisante ?}

Non ce n'est pas possible car si on met le seuil trop haut afin d'éliminer des pixels "bruit" on élimine par la même occasion des pixels de contour (\emph{cf.}~Fig.~\ref{seuillage} à droite). Mais si le seuil est trop bas on garde trop de pixel "bruit" (\emph{cf.}~Fig.~\ref{seuillage} à gauche).

On voit bien que cette approche n'est pas satisfaisante il faudrait avoir plusieurs seuils en fonction de la position du pixel. Il faut pour cela calculer des maxima locaux à l'image de la norme du gradient.

\section{Seuillage des maxima locaux par hystérésis}

Dans cette partie nous allons expliquer l'algorithme du seuillage des maxima locaux par hystérésis grâce à l'algorithme suivant:

\begin{Verbatim}[commandchars=\\\{\}]
public ByteProcessor \codeRed{hystIter}(ImageProcessor imNormeG, int seuilBas, int seuilHaut) \{
        
	\codeBlue{// On récupère la taille de l'image}
	int width = imNormeG.getWidth();
	int height = imNormeG.getHeight();

	\codeBlue{// On va créer une liste de pixels contours pour lesquels la valeur}
	\codeBlue{// est supérieure au seuil haut.}
	\codeBlue{// cette liste servira à savoir si il y a encore des pixels voisins}
	\codeBlue{// aux pixels contours compris entre les deux seuils}
	ByteProcessor maxLoc = new ByteProcessor(width,height);
	List<int[]> highpixels = new ArrayList<int[]>();

	for (int y=0; y<height; y++) \{
		for (int x=0; x<width; x++) \{

			\codeBlue{// On récupère le pixel courant de l'image issue}
			\codeBlue{// du calcul des maxima locaux}
			int g = imNormeG.getPixel(x, y)&0xFF;
                
			\codeBlue{// Si le pixel est inférieur au seuil bas on laisse le pixel à 0.}
			\codeBlue{// Cela permet d'éliminer les pixels non contour (non candidat).}
			if (g<seuilBas) continue;

			\codeBlue{// Si le pixel est supérieur au seuil haut on}
			\codeBlue{// le garde comme pixel contour.}
			if (g>seuilHaut) \{
				maxLoc.set(x,y,255);
				highpixels.add(new int[]\{x,y\});
				continue;
			\}

			\codeBlue{// On met le pixel dont la valeur est comprise entre le seuil bas}
			\codeBlue{// et le seuil haut dans la nouvelle image. Ce seront les pixels}
			\codeBlue{candidats.}
			maxLoc.set(x,y,128);
		\}
	\}


	\codeBlue{// on sauvegarde les pas pour accéder aux pixels voisins}
	int[] dx8 = new int[] \{-1, 0, 1,-1, 1,-1, 0, 1\};
	int[] dy8 = new int[] \{-1,-1,-1, 0, 0, 1, 1, 1\};
        
	\codeBlue{// La nouvelle liste de pixels contours}
	List<int[]> newhighpixels = new ArrayList<int[]>();

	\codeBlue{// tant qu'il reste des candidats potentiels}
	while(!highpixels.isEmpty()) \{
        	
		\codeBlue{// on vide la liste de nouveaux pixels contours}
		newhighpixels.clear();
            
		\codeBlue{// Pour chaque pixel coutour}
		for(int[] pixel : highpixels) \{
			int x=pixel[0], y=pixel[1];
                
			\codeBlue{// On cherche à savoir lesquels de ses voisins}
			\codeBlue{// sont des pixels candidats.}
			\codeBlue{// Si ce pixel est un pixel candidat alors on l'ajoute}
			\codeBlue{// dans les nouveaux pixels contours.}
			for(int k=0;k<8;k++) \{
				\codeBlue{// On récupère le voisin en fonction du pas}
				int xk=x+dx8[k], yk=y+dy8[k];
                    
				\codeBlue{// on évite les effets de bord}
				if (xk<0 || xk>=width) continue;
				if (yk<0 || yk>=height) continue;
                    
				\codeBlue{// si le pixel est un candidat on le sauvegarde en tant que}
				\codeBlue{// nouveau pixel contour et on met la valeur à l'indice de ce}
				\codeBlue{// pixel à 255}
				if (maxLoc.get(xk, yk)==128) \{
					maxLoc.set(xk, yk, 255);
				newhighpixels.add(new int[]\{xk, yk\});
				\}
			\}
		\}

		\codeBlue{// On met dans la nouvelle liste de pixels contours les nouveaux pixels}
		\codeBlue{// précédemment calculé}
		List<int[]> swap = highpixels; 
		highpixels = newhighpixels; 
		newhighpixels = swap;
	\}

	\codeBlue{// On élimine les pixels isolés}
	for (int y=0; y<height; y++) \{
		for (int x=0; x<width; x++) \{
			if (maxLoc.get(x, y)!=255) maxLoc.set(x,y,0);
		\}
	\}
        
	\codeBlue{// On retourne l'image}
	return maxLoc;
\}
\end{Verbatim}

Lorsque cet algorithme est appelé sur l'image résultant du calcul des maxima locaux (\emph{cf.} Fig.~\ref{norme_max_locaux} à droite) on obtient la Fig.~\ref{seuillage_hysteresis}

\section*{Conclusion}

On se rend bien compte que la détection de contour n'est pas une chose simple. Il faut calculer la norme de gradient de l'image pour ensuite en tirer une image résultant du calcul des maxima locaux et enfin appliquer un seuillage par hystérésis. Après tous ces calculs on obtient une image de contour satisfaisante mais malgré tout il reste des contours indésirables. Il faudra peut être appliquer de nouveaux calculs sur l'image afin d'enlever tout contour indésirable.
